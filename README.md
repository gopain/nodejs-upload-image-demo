nodejs-upload-image-demo
========================

1. 初始化项目信息：npm init
2. 修改package.json，配置启动命令：

  "scripts": {
      "start": "node index.js"
  },

3. 安装依赖模块：npm install formidable --save
4. index.js, server.js, router.js, requestHandler.js

参考：

    http://www.nodebeginner.org/index-zh-cn.html#handling-file-uploads
